# Notices

SignalHarness is an independent project inspired by general agent harness design patterns.
It does not vendor or depend on OpenHarness code.

The project borrows common engineering ideas used across modern Agent systems:
structured outputs, controlled tool loops, permission guardrails, local trace
records, deterministic fallback, and review-only learning proposals. The
implementation, docs, fixtures, tests, and CLI are maintained as SignalHarness
project code.
